## 注意事项

1. 本项目仅用于方便更多开发者体验ChatGPT,生成的内容与本平台无关
2. 项目仅供一乐，采用个人token实现
3. 对话记忆时长默认设置10分钟，可以自行在代码第18行更改。
4. 有问题可以在评论区讨论
5. 好玩的对话也可以放评论区大家看看，但是禁止搞zz、ys和任何敏感问题。
6. 如果想自行更改token，可以查看TOKEN.md进行操作
